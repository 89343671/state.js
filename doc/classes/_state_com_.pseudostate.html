Class PseudoState

An element within a state machine model that represents an transitory Vertex within the state machine model.

Pseudo states are required in all state machine models; at the very least, an Initial pseudo state is the default stating state when the parent region is entered. Other types of pseudo state are available; typically for defining history semantics or to facilitate more complex transitions. A Terminate pseudo state kind is also available to immediately terminate processing within the entire state machine instance.

PseudoState extends the Vertex class and inherits its public interface.
