Enumeration PseudoStateKind

An enumeration of static constants that dictates the precise behavior of pseudo states.

Use these constants as the kind parameter when creating new PseudoState instances.

Choice

Choice:
  • Defined in state.com.ts:49

Enables a dynamic conditional branches; within a compound transition. All outbound transition guards from a Choice are evaluated upon entering the PseudoState: if a single transition is found, it will be traversed; if many transitions are found, an arbitary one will be selected and traversed; if none evaluate true, and there is no 'else transition' defined, the machine is deemed illformed and an exception will be thrown.
