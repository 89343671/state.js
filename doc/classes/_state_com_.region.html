Class Region

An element within a state machine model that is a container of Vertices.

Regions are implicitly inserted into composite state machines as a container for vertices. They only need to be explicitly defined if orthogonal states are required.

Region extends the Element class and inherits its public interface.

Static defaultName

defaultName: string = "default"
  • Defined in state.com.ts:183

The name given to regions that are are created automatically when a state is passed as a vertex's parent. Regions are automatically inserted into state machine models as the composite structure is built; they are named using this static member. Update this static member to use a different name for default regions.
